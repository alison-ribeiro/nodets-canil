# nodets-canil
    Projeto feito no módulo do cruso Node + Typescript
### Pré-requisitos globais:

`npm i -g nodemon typescript ts-node` 
ou
`yarn global add nodemon typescript ts-node`  

### Instalação

`npm install`
ou
`yarn install`

## Para rodar o projeto

`npm run start-dev`
ou
`yarn run start-dev`
